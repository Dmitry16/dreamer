# UX Flow – Minimum Lovable Product

## 1. Home / Dashboard
- Primary action: "Record a Dream"
- Secondary: Dream history list

---

## 2. Dream Entry
- Large text area for dream narration
- Optional fields: mood, life context
- Continue button (no interpretation yet)

---

## 3. Dream Breakdown
System presents extracted elements as cards:
- Characters
- Symbols
- Places
- Emotions

User can:
- Edit labels
- Remove irrelevant items
- Add missing elements

---

## 4. Associations Screen
For each symbol card:
- "What does this remind you of?"
- "How do you feel about it?"
- "Where does this show up in your life?"

Progress indicator to encourage completion.

---

## 5. Interpretation Screen
- 2–3 hypotheses displayed as expandable cards
- Each card shows:
  - Jungian lens
  - Evidence from dream
  - Association references
  - Reflective question
- Buttons: "Resonates" / "Doesn't fit"

---

## 6. Integration
- Short reflective summary
- 1–2 integration prompts
- Optional journal input

---

## 7. Save & Return
- Dream saved to history
- Return to dashboard
